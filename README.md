# Escape Room — Programação

# Quiz da Copa do Mundo — Estádio Secreto

> Atividade Final — Gamificação | Lógica de Programação

---

## História do Jogo

Você foi trancado no **Estádio Secreto da FIFA**!  
A única saída é responder **8 perguntas** sobre a Copa do Mundo corretamente.  
Cada acerto vale **10 pontos**. Você consegue escapar?

---

## Como Executar

1. Abra o arquivo quiz.c no Dev-C++
2. Compile o programa
3. Execute o jogo
4. Responda as 8 perguntas para fugir do estadio.

---

## Exemplo de Execução

```
=====================================================
     QUIZ DA COPA DO MUNDO - ESTADIO SECRETO
=====================================================

HISTORIA:
Voce foi trancado no estadio secreto da FIFA!
A unica saida e responder 8 perguntas sobre a
Copa do Mundo corretamente.
Cada acerto vale 10 pontos.

-----------------------------------------------------

[ Pergunta 1 de 8 ]
Qual selecao ganhou a Copa do Mundo de 2022 no Qatar?

  A) Brasil
  B) Franca
  C) Argentina
  D) Alemanha

Sua resposta (A/B/C/D): C

✓ Correto! Voce avancou no estadio!
```

## Conceitos de Programação Utilizados
